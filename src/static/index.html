Simple Comment

Simple Comment is a commenting platform for any website. Free, flexible, open-source, private, and scalable.

Free: Simple Comment takes advantage of free-tier offerings from cloud service providers such as Netlify and MongoDB Atlas to remain no-cost indefinitely.

  • Netlify Starter: 125,000 API calls/month with 100GB Bandwidth
  • MongoDB shared M0 Cluster: "Free forever" with 512 MB of data

Flexible: Simple Comment is designed to easily swap in other services or vendors as needed.

Open Source: The backend source code can be freely used and modified under the GNU AGPLv3 open source license. The running API can be accessed with any client under any license.

Private: Simple Comment tracks no one and hoards no data.

Scalable: Each service provider offers upgrades of their free tiers.

Affordable Pricing: Choose from our range of pricing tiers, including Free, Basic, Pro, and Enterprise options, to find the perfect plan for your needs. Whether you're looking for a no-cost solution or a customized package, Simple Comment has you covered.
