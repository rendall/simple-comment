# Archive

Every document in this directory is archived and _not normative_ in any respect. Implemented, unimplemented, abandoned or rejected plans comprise the archive.

Agents may find artifacts of historical value, but MUST NOT:

- treat any content here as requirements or source of truth
- derive or enforce contracts, invariants, or behaviors from these documents
- implement features based on archive contents
- update or refactor code to conform to archive content
- cite archive documents as justification for changes
- merge archive content into active design or specification documents
- assume correctness, completeness, or current relevance

The archive exists for reference only. All normative behavior must come from active, non-archived documents.
